Work

Part 1: RF, SD, and CG in Eigen Coordinate System

  • Matrix generation with controlled eigenvalue distributions
  • Implementation of RF, SD, and CG methods
  • Performance analysis and comparison
  • Convergence analysis

Part 2: Stationary Methods

  • Unified implementation of Jacobi, Gauss-Seidel, and Symmetric Gauss-Seidel
  • Spectral radius and matrix norm calculations
  • Convergence analysis

Part 3: Sparse Matrix Implementation

  • Sparse matrix storage and operations
  • Efficient implementations of all methods for sparse matrices
  • Performance analysis with varying sparsity

Getting Started

Prerequisites

  • Python 3.8 or higher
  • NumPy
  • SciPy
  • Matplotlib
  • Jupyter Notebook (for analysis notebooks)

Installation

git clone https://github.com/yourusername/numerical-methods.git
cd numerical-methods
pip install -r requirements.txt
