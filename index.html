Overview

This project implements and analyzes various numerical methods for solving linear systems, including:

  • Richardson's First Order Stationary method (RF)
  • Steepest Descent (SD)
  • Conjugate Gradient (CG)
  • Jacobi Method
  • Gauss-Seidel Method
  • Symmetric Gauss-Seidel Method

Part 1: RF, SD, and CG in Eigen Coordinate System

Implementation Details

  • Matrix generation with controlled eigenvalue distributions
  • Implementation of RF, SD, and CG methods
  • Performance analysis and comparison
  • Convergence analysis

Key Files

  • rf.py: Richardson's First Order Stationary method implementation
  • sd.py: Steepest Descent method implementation
  • cg.py: Conjugate Gradient method implementation
  • eigen_analysis.py: Tools for eigenvalue analysis

Part 2: Stationary Methods

Implementation Details

  • Unified implementation of Jacobi, Gauss-Seidel, and Symmetric Gauss-Seidel
  • Spectral radius and matrix norm calculations
  • Convergence analysis

Key Files

  • stationary_methods.py: Implementation of all stationary methods
  • matrix_analysis.py: Tools for matrix analysis

Part 3: Sparse Matrix Implementation

Implementation Details

  • Sparse matrix storage and operations
  • Efficient implementations of all methods for sparse matrices
  • Performance analysis with varying sparsity

Key Files

  • sparse_operations.py: Core sparse matrix operations
  • sparse_solvers.py: Sparse implementations of all solvers

Getting Started

Prerequisites

  • Python 3.8 or higher
  • NumPy
  • SciPy
  • Matplotlib
  • Jupyter Notebook (for analysis notebooks)

Installation

git clone https://github.com/yourusername/numerical-methods.git
cd numerical-methods
pip install -r requirements.txt
