Getting Started

Prerequisites

  • Python 3.8 or higher
  • NumPy
  • SciPy
  • Matplotlib
  • Jupyter Notebook (for analysis notebooks)

Installation

git clone https://github.com/yourusername/numerical-methods.git
cd numerical-methods
pip install -r requirements.txt
